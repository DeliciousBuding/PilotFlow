# 演示材料

## 演示脚本

### 场景：一句话创建项目空间

**演示步骤：**

1. 打开飞书群聊
2. 输入：`@PilotFlow 帮我准备答辩项目空间，成员delicious233，交付物是项目简报和任务清单，5月7日截止`
3. PilotFlow 返回执行计划卡片
4. 点击「确认执行」
5. PilotFlow 自动创建：文档、多维表格记录、任务、项目入口消息
6. PilotFlow 发送交付总结

**预期时间：** 2 分钟

### 场景：风险检测

**演示步骤：**

1. 输入：`@PilotFlow 创建一个新项目`
2. PilotFlow 提示信息不完整，询问目标、成员、交付物
3. 补充信息后，PilotFlow 识别风险并展示
4. 确认后执行
